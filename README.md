# Pharmacy-Management-System
A JavaFX-based Pharmacy Management System with MySQL database for managing medicines, suppliers, employees, orders, and invoices.

Pharmacy Management System:
A desktop-based Pharmacy Management System developed using Java, JavaFX, and MySQL.
The system is designed to manage pharmacy operations efficiently, including medicine inventory, suppliers, employees, orders, and invoices.

Features:
Manage medicines (Add, Update, Delete, View)
Supplier management
Employee management
Order and order details handling
Invoice generation
Editable TableView using JavaFX
MySQL database integration
User-friendly JavaFX interface

Technologies Used
Java
JavaFX
MySQL
JDBC
Scene Builder

Database Structure
The system includes the following main tables:
Medicines
Suppliers
Employees
Orders

OrderDetails

Invoices
